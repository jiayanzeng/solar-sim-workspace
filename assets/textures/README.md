# Surface texture assets

Every shipped `.ktx2` has a sibling `.license.json` that records the exact
NASA or USGS source URL, source SHA-256, dimensions, public-domain policy
route, transformation, and shipped-asset SHA-256. Run
`scripts/check-texture-metadata.sh` to validate the complete directory.

The images are public-domain U.S. Government works. Their inclusion is data
provenance, not NASA endorsement; no NASA logo, insignia, or brand treatment
is included. Source rasters are not redistributed. The KTX2 files are 2K
runtime derivatives generated by `xtask convert-texture` from curated PPM
staging files. Bodies without a curated global map intentionally keep their
catalog color rather than receiving an invented texture.
