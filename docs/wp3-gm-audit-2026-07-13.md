## Source and convention

The binding marker asks for comparison with JPL DE440. The primary source is
Table 2 of Park et al., [The JPL Planetary and Lunar Ephemerides DE440 and
DE441](https://ssd.jpl.nasa.gov/doc/de440_de441.html). JPL SSD publishes the
same numbers in its [Astrodynamic Parameters](https://ssd.jpl.nasa.gov/astro_par.html)
table.

Use the exact DE440 values in km³/s². Mercury, Venus, and Earth are body GMs;
Mars through Neptune are system GMs in DE440. That distinction is compatible
with the approved orbit routing: Mercury–Mars still use geometric-center
Horizons targets, while Jupiter–Neptune use system-barycenter targets. A GM
does not change a generated planet's own heliocentric state; it is the parent
parameter used by the runtime's two-body fallback, including moon propagation.
The source table below preserves every published decimal; Rust stores each as
the nearest representable `f64` (the solar literal therefore prints as
132,712,440,041.27942).
